RollDash(最終更新日2017/6/5)

この度はRollDashに興味を持ってくださり誠にありがとうございます。

【ソフト名】
　RollDash

【種別】
　玉転がしゲーム

【ゲーム概要】
　矢印マークの書かれたボールを左右に動かし障害物を避けたりステージに散りばめられた仕掛けを使いゴールを目指すゲームです。
　
　青い板がジャンプ板で、矢印の描いてある板が加速板、黄色い板が減速板となっております。

【起動方法】
　RollDash.exeを起動したあとにGameStartの文字をクリックするとゲームが始まります。

【終了方法】
　画面右上の×ボタンを押してください

【対応操作】
　マウス及びマウスパッドと左右キーのみとなります。

【動作環境】
　Windows 8.1,10での動作を確認

【開発環境】
　Unity　C#

【開発期間】
　2日

【工夫点】
　敵が追いかけてくるスピードや障害物の配置をクリアできるかできないかのギリギリに設置してゲームの難易度を調整しました。

【改善点】
  クリアスピードを競えるようにタイマー機能の追加およびランキング画面を実装をしてよりゲーム性を高めたいと考えています。
